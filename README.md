# Linux & Git Fundamentals Quick Start Guide

This repository serves as a comprehensive, production-ready reference guide for essential Linux CLI commands and GitHub integration workflows. 

## 🗺️ Roadmap & Contents
Inside this repository, you will find detailed guides on:
1. **File & Directory Management:** `touch`, `mkdir`, `cp`, `mv`.
2. **Text Viewing & Inspection:** `cat`, `more`, `less`, `head`, `tail`, `multitail`.
3. **Data Pipelines:** Mastering the Linux Pipe (`|`) operator.
4. **CLI Text Editors:** Comprehensive comparison and usage of `vi/vim` and `nano`.
5. **GitHub Integration:** Step-by-step authentication via HTTPS (Personal Access Tokens) and SSH Keys.
